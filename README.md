# 📊 쇼핑몰 자동화 3단계 쇼케이스

> **같은 문제도 예산에 맞게 3가지 수준으로 해결합니다.**
> 이 저장소는 실제로 동작하는 3단계 자동화를 모두 담고 있습니다 — 필요한 단계만 골라 의뢰하세요.

## 한눈에 보기

| | Level 1 · 기본 | Level 2 · 통합 파이프라인 | Level 3 · AI 인사이트 |
|---|---|---|---|
| **해결하는 문제** | 매일 반복하는 엑셀 정리 | 채널마다 다른 형식의 데이터 통합 | "데이터는 있는데 뭘 해야 할지 모름" |
| **제공 기능** | 일일 매출·재고 엑셀 리포트 자동 생성 | 다중 소스 수집→검증→병합→리포트→알림 | 이상 감지·재고 소진 예측·AI 경영 코멘트·웹 대시보드 |
| **적합한 곳** | 1인 셀러, 단일 채널 | 다채널 운영 (스토어+쿠팡+자사몰) | 데이터로 의사결정하려는 팀 |
| **작업 기간** | 3~5일 | 7~10일 | 2~3주 |
| **참고 가격대** | 30만원~ | 80만원~ | 150만원~ |

각 레벨은 독립적으로 실행 가능하며, 하위 레벨 위에 쌓아 올리는 구조입니다 — **처음엔 Level 1로 시작하고 사업이 크면 업그레이드**하는 것도 가능합니다.

## Level 1 — 일일 엑셀 리포트 자동화 (저장소 루트)

매일 아침 어제의 매출·재고 현황이 엑셀로 자동 생성됩니다. 수작업 2~3시간 → 0분.

```bash
pip install pandas openpyxl
python generate_sample_data.py   # 데모용 샘플 데이터
python report_generator.py       # → output/일일리포트_YYYY-MM-DD.xlsx
```

리포트 구성: 일일 요약(전일 대비 증감), 상품 TOP10+차트, 재고 경고+발주 추천 수량, 14일 추이+차트. 작업 스케줄러/cron 등록으로 완전 자동화, Slack 웹훅 알림 지원.

## Level 2 — 다중 채널 통합 파이프라인 (`level2_통합파이프라인/`)

실무의 진짜 문제: 스마트스토어·쿠팡·자사몰이 **각각 다른 형식**의 데이터를 줍니다. Level 2는 이를 설정 파일 하나로 통합합니다.

```bash
cd level2_통합파이프라인
python make_source_data.py   # 형식이 서로 다른 3개 채널 데이터 생성 (더러운 데이터 포함)
python pipeline.py           # 수집→정규화→검증→병합→리포트까지 한 번에
```

Level 1과의 차이:

- **데이터 검증**: 중복 주문 제거, 금액 불일치 자동 보정, 결측 행 격리 — 처리 내역이 `pipeline.log`에 전부 기록됩니다
- **자동 재시도**: 일시적 오류(파일 잠김 등)에 3회 재시도
- **확장성**: 새 판매 채널 추가 = 코드 수정 없이 `config.json`에 컬럼 매핑만 추가

## Level 3 — AI 인사이트 대시보드 (`level3_AI인사이트/`)

집계를 넘어 **판단을 자동화**합니다. "그래서 오늘 뭘 해야 하는데?"에 답합니다.

```bash
cd level3_AI인사이트
python insights.py           # → dashboard.html 생성 (브라우저로 열기)
```

- **매출 이상 감지**: 주간 평균 대비 급등/급락 자동 경보
- **재고 소진 예측**: 판매 속도 기반 D-day 계산 → "지금 발주하세요" 알림
- **상품 추세**: 뜨는 상품/지는 상품 자동 분류
- **AI 경영 코멘트**: `ANTHROPIC_API_KEY` 설정 시 Claude가 데이터를 읽고 실행 가능한 조언 생성 (키 없이는 규칙 기반 코멘트로 동작)
- **웹 대시보드**: 차트·KPI·인사이트가 담긴 HTML 자동 생성
